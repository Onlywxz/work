# 技术分析写作规则

1. 每日技术分析必须使用中文。
2. 分析风格要接近以下结构：
   - 先判断当前图形所处阶段，而不是直接给结论；
   - 区分：趋势结构、均线结构、MACD、BOLL、KDJ/RSI；
   - 不要机械罗列指标数值；
   - 重点解释指标背后的市场含义；
   - 必须给出后续可能路径；
   - 必须给出关键观察位；
   - 必须给出应对方案；
   - 必须说明当前判断的前提和失效条件。
3. 输出语气要求：
   - 专业、克制、清晰；
   - 不要夸张；
   - 不要直接喊多或喊空；
   - 要像投研日报，不像交易喊单。
4. 每日总结固定输出到 `reports/daily-summary.md`。
5. 新总结追加到文件顶部，保留历史内容。
