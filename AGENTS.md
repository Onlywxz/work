# AGENTS.md

这是一个生猪相关的日更工作流。

我会上传已经制作好的本地 HTML 文件。

需要完成的事情：

1. 每天抓取当天的生猪现货均价。
2. 将当天现货均价作为当天现货 K 线的收盘价。
3. 当天现货 K 线的开盘价使用前一天现货 K 线的收盘价。
4. 更新 HTML 中最新的现货 K 线。
5. 根据 HTML 图中已有的技术指标做技术分析。
6. 将当日总结输出到一个固定的日更文件中。
7. 日更文件每天必须标明日期。
8. 每天抓取大连商品交易所所有生猪合约的收盘价。
9. 结合当日现货均价，计算各个生猪合约的基差。
10. 生成基差折线图。
11. 基差折线图横轴为日期。
12. 基差折线图纵轴为基差数据。
13. 不同合约使用不同颜色的线。
14. 图中必须标明图例。
15. 基差图风格参考用户提供的图片。

不要自行扩展需求。
不要自行假设其他项目背景。
用户说什么，就按什么做。
